# ONabat Logo

## Download Logo

Please download the official ONabat logo from:
[https://share.icloud.com/photos/05b2OQFlv5lYFnJHoHDfqgfcg](https://share.icloud.com/photos/05b2OQFlv5lYFnJHoHDfqgfcg)

## Recommended File Names

After downloading, save the logo in this directory with appropriate names:

- `onabat-logo.png` - Standard PNG format
- `onabat-logo.svg` - Vector format (if available)
- `onabat-logo-dark.png` - Dark theme variant (if needed)
- `onabat-logo-light.png` - Light theme variant (if needed)

## Logo Specifications

Place your logo files in this directory following these specifications:

- **Format**: PNG, SVG, or both
- **Transparency**: Recommended for flexible usage
- **Resolution**: High resolution (at least 1024x1024px for PNG)
- **Aspect Ratio**: Maintain original proportions

## Usage

Once downloaded, the logo can be referenced in documentation and applications from this path:
```
assets/branding/logos/onabat-logo.png
```
