# ONabat Branding Assets

This directory contains branding assets for the ONabat project.

## Logo

The official ONabat logo can be accessed at:
- **Source**: [iCloud Photo Link](https://share.icloud.com/photos/05b2OQFlv5lYFnJHoHDfqgfcg)

Please download the logo from the link above and place it in the `logos/` directory.

### Logo Usage Guidelines

- Use the official logo as provided
- Maintain proper spacing and proportions
- Do not modify or distort the logo
- Ensure proper contrast with backgrounds
